Ecco Text Generator, Beta Version 0.99.0428

Created by Johnny L. de Alba (Arkonviox), 2020.

What is this?

It's an Ecco the Dolphin Text Generator that creates animated gifs and static images using the ripple generating algorythm from Ecco 2: The Tides of Time.
